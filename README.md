# Event Booking Manager Pro + TPOTS Custom Theme

A production-grade WordPress solution that includes:

- Custom Event Booking Plugin (MVC architecture)
- Dynamic Gutenberg Block (Upcoming Events – SSR)
- Performance-optimized Custom Theme (TPOTS)
- REST API integration
- Clean scalable architecture

---

## 🚀 Features

### Plugin (Event Booking Manager Pro)

- Custom Post Type: Events
- MVC based architecture
- REST API endpoint for events
- Server Side Rendered Gutenberg Block
- Service layer abstraction
- Composer-based structure

### Theme (TPOTS)

- Lightweight performance optimized theme
- Clean header/footer architecture
- Minimal CSS for fast rendering
- WordPress standards compliant

### Gutenberg Block

- Dynamic upcoming events listing
- SSR rendering for performance
- Block.json based modern structure
